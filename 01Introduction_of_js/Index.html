Introduction 0f javaScript

  • javaScript light weight event based programming language hai jo Website or Webpage ko dynamic interactive banane ke liye use hoti hai,
    or js ko HTML document me use karte hai.
  • javaScript ko Netscape communication company ne banaya tha [Brendan Eich - 1995]
    Current - Oracle corporation company ke pas hai.

Features of javaScript

  • js sabhi browser ko support karti hai.
  • javaScript C programming language ke structure or syntax ko follow karti hai.
  • Light weight hai kam storage use karti hai.
  • case-sensitive hai
  • synchronous single threaded language hai
  • Web browser pe user ko achha control provide karati hai
