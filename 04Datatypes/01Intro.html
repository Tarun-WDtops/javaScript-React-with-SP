Data Type Inroduction

DataType alag alag type ke data ko describe karta hai, Jisko use karke ham ise variable me store kara sakte hai.

JavaScript has 2 Datatypes

1. Primitive Datatypes

  • Primitive Datatypes wo hote hai jisme ek time mein ek hi type ke datatype ki value run ho sakti hai.

    • 2. Non-Primitive Datatypes

  • Non-Primitive Datatypes wo datatypes hote jisme ham multiple value run kara sakte hai.

    • 1. Primitive Datatypes

    > There are 7 types of Primitive Datatypes -

    1. String

    2. Number

    3. Null

    4. Boolean

    5. Undefined

    6. Bigint

    7. Symbol

    > There are 2 types of Non-Primitive Datatypes -

    1. Arrey

    2. Object
